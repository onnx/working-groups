## Papers Reviewed / Under Consideration

### Counter-Based RNG Design

**Parallel Random Numbers: As Easy as 1, 2, 3**  
John K. Salmon, Mark A. Moraes, Ron O. Dror, David E. Shaw (SC11)

This paper introduces counter-based pseudo-random number generators, which eliminate the sequential dependence found in traditional PRNGs. Instead of computing the next random number from the previous state, the generator computes the n-th value directly from a counter.

The design makes random number generation:

* naturally parallelizable
* deterministic across distributed systems
* efficient on GPUs and HPC systems

The algorithms introduced in this work (including Threefry and Philox) are foundational to the RNG approach used in JAX and other modern ML systems.

### Vectorized and Hardware-Accelerated MCMC

**Efficiently Vectorized MCMC on Modern Accelerators**

This paper explores how vectorization tools such as `vmap` in JAX enable large-scale parallel execution of MCMC chains on modern hardware accelerators.

The authors show that naive vectorization can introduce synchronization bottlenecks across chains and propose algorithmic restructuring techniques (such as finite state machine formulations) to avoid these issues.

---

### Running MCMC on Modern Hardware

**Running Markov Chain Monte Carlo on Modern Hardware and Software**

This work examines how modern machine learning frameworks such as JAX and PyTorch can be used to accelerate Bayesian inference workloads using GPUs and TPUs.

The paper highlights the importance of:

* vectorized computation
* automatic differentiation
* hardware-aware algorithm design

for scaling MCMC methods to modern compute architectures.
